- Tarefa: P1 - Desenvolver Tela de Jornadas

  - Porquê: Adicionar uma funcionalidade central de gestão de viagens.
  - Complexidade: Média
  - Concluído: [ ]

  - Porquê: Estabelecer a base para o desenvolvimento da tela de jornadas.

  - Concluído: [ ]

  - Quem: Frontend

  - Complexidade: Média
  - Concluído: [ ]
